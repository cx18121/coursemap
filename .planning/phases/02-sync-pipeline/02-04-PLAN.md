<!-- API response shapes from Plan 03 that the UI consumes -->

GET /api/parse-ics response:
```typescript
{
  courses: Array<{
    courseName: string;
    colorId: string;
    enabled: boolean;
    events: Array<{
      uid: string;
      summary: string;      // original Canvas title
      cleanedTitle: string;  // AI-cleaned or regex-cleaned
      description: string;
      start: string;         // ISO date
      end: string;           // ISO date
      excluded: boolean;     // true if user excluded this event
    }>;
  }>;
}
```

GET /api/school-calendars response:
```typescript
{
  calendars: Array<{ calendarId: string; name: string; selected: boolean; }>;
  linked: boolean;  // false if no school account
}
```

PUT /api/user-selections body:
```typescript
{
  courseSelections?: Array<{ courseName: string; enabled?: boolean; colorId?: string; }>;
  eventOverrides?: Array<{ eventUid: string; enabled?: boolean; customTitle?: string; }>;
}
```

PUT /api/school-calendars body:
```typescript
{ calendarId: string; calendarName: string; enabled: boolean; }
```

POST /api/sync response:
```typescript
{ jobId: string }  // 202 Accepted
```

GET /api/sync/status?jobId=X response:
```typescript
{
  status: 'running' | 'complete' | 'error';
  progress: Array<{ courseName: string; processed: number; total: number; }>;
  canvasSummary?: { inserted: number; updated: number; skipped: number; failed: number; errors: string[]; };
  mirrorSummary?: { inserted: number; updated: number; skipped: number; failed: number; errors: string[]; };
  error?: string;
}
```

- CSS variables: --color-text-primary, --color-text-secondary, --color-surface, --color-border
- Server Component reads session -> passes props to 'use client' children
- useState for local state, no global state manager
